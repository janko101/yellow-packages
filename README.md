## Description
This is a simple application for tracking packages

## Getting started
### Dependencies  
* Npm
* React
* Axios

### Installation and Setup   
Clone down this repository. You will need node and npm installed globally on your machine. </br>

Install all of the dependencies:    
```
$ npm install
```  

To Start Server:    
```
$ npm start
```
